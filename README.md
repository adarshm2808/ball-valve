# Ball Valve Assembly Design
Ball Valve Assembly Design :Designed and modeled a complete ball valve assembly using 3D CAD software.

## Overview

This project focuses on the design and 3D modeling of a mechanical ball valve assembly using SolidWorks. The model demonstrates the working mechanism of a quarter-turn flow control valve commonly used in piping and fluid systems.

The assembly was created to understand component interaction, rotational motion, and mechanical design principles involved in industrial valve systems.


## Features

* Complete 3D ball valve assembly model
* Rotating handle mechanism for flow control
* Internal flow passage design
* Detailed component modeling and assembly
* Realistic product visualization
* Engineering-oriented mechanical structure


## Components Modeled

* Hndle
* Ball Element
* Lid
* Slide



## Software Used

* SolidWorks



## Design Process

1. Created individual part sketches and 3D features.
2. Developed the valve body with internal flow passage.
3. Modeled the rotating ball and stem connection.
4. Designed ergonomic handle for valve actuation.
5. Assembled all components using mates and constraints.
6. Performed fit and interference checks.
